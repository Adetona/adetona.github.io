WP Medium Editor gives you a new writing experience on wordpress.

Quick start.

Via WordPress site Dashboard.

  • Plugin > Add New > Search 'WP Medium Editor' > Install > Activate.

    • Manual Installation.

  • Download the Zip file

  • Upload the entire `WP-Medium-Editor` folder to the `/wp-content/plugins/` directory.

  • Activate the plugin through the 'Plugins' menu in WordPress.
